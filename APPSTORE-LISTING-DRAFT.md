## アプリ名（Listing name）

- 造語のため、ガイドラインの「一般名詞のみのアプリ名は不可」に抵触しない。
- 「カメラのレンズでAmazonの相場を覗く」という比喩で、スキャンして価格を見るという機能と合致する。
- サブタイトル（App Store用）: スキャンで価格・利益・グラフを1画面
- 代替案: アマソク / せどリサーチ（採用しない場合の予備。せどリサーチは一般名詞のみのためブランド名を冠する必要あり）。
- ウェブサイト sellira.jp 側にも「アマレンズ」の名称とロゴを掲載すること（ガイドラインでサイトとアプリ名の一致が求められるため）。
- iOSアプリの表示名（CFBundleDisplayName）も「アマレンズ」に統一済み。

### Long description（4000字以内・現在約900字）

```
アマレンズは、日本のAmazonストアで書籍を販売するセラー向けの仕入れリサーチ用iOSアプリです。店頭で本のバーコード（ISBN・JAN）をスキャンすると、Amazonでの新品・中古の価格や売れ筋ランキングを素早く確認できます。

主な仕組み

・バーコードスキャン：カメラで本のバーコードを読み取り、商品を特定します。バーコードが読み取れない本では、OCR（文字認識）でISBNやJANの印字を読み取ることもできます。

・商品情報と価格の確認：Selling Partner APIのCatalog Items APIで商品情報（タイトル・画像・売れ筋ランキング）を取得し、Product Pricing APIで新品・中古のオファー情報（コンディション・送料込み価格）を一覧表示します。

・損益分岐点の目安：Product Fees APIの手数料見積もりを用いて、販売手数料を差し引いた損益分岐点の目安を表示し、その場での仕入れ判断を支援します。

Amazonアカウントとの連携（OAuth）

出品者向け機能は、販売パートナーご自身のAmazonアカウントをOAuthで連携して利用します。連携により取得したデータは、その販売パートナー本人の仕入れ判断のためだけに使用します。第三者への共有は行いません。購入者の個人情報（PII）にはアクセスしません。

料金

基本機能は無料でご利用いただけます。すべての機能を制限なく利用できる有料プラン（月額1980円・3日間の無料体験つき）を提供しています。詳細はウェブサイトをご覧ください。

動作環境

・iOS 16以上のiPhone
・日本のAmazonストア（amazon.co.jp）のみ対応
```


### Who should use this（1000字以内）

```
・日本のAmazonストア（amazon.co.jp）で中古書籍などを販売しているセラー向けのアプリです。
・店頭での仕入れの際に、その場で価格と利益の目安を確認したい方に適しています。
・利用にはiOS 16以上のiPhoneが必要です。
・出品者向け機能（オファー一覧・損益分岐点の目安）の利用には、Amazonの大口出品アカウントとSelling Partner APIの連携（OAuth）が必要です。
・日本のAmazonストアのみに対応しています。他の国のストアには対応していません。
```
